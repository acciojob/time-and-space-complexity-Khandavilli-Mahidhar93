Shortnote on Time, Space, Complexity

Time complexity is measure of how long an algorithm takes to run without looking to size of input. The algorithms with low time complexity has most efficiency.

Space Complexity of a function is determined by the amount of memory it uses. So, when algorithm performs computation on each item it takes O(n) time.
